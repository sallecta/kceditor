KCEditor Samples » Append To Page Element Using JavaScript Code

The kceditor.appendTo() method serves to to place editors inside existing DOM elements. Unlike kceditor.replace(), a target container to be replaced is no longer necessary. A new editor instance is inserted directly wherever it is desired. 

kceditor.appendTo( 'container_id',
	{ /* Configuration options to be used. */ }
	'Editor content to be used.'
);
