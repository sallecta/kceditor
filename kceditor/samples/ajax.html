KCEditor Samples » Create and Destroy Editor Instances for Ajax Applications

This sample shows how to create and destroy KCEditor instances on the fly. After the removal of KCEditor the content created inside the editing area will be displayed in a <div> element.

For details of how to create this setup check the source code of this sample page for JavaScript code responsible for the creation and destruction of a KCEditor instance.

Click the buttons to create and remove a KCEditor instance.
